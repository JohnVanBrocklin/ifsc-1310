    Thoughts on Progressive Enhancement

    Progressive Enhancement is the progress of making sure that things are done in order to ensure stability. An example of this is making sure that a machine displays in order from html, css, and then js, and makes sure that it loads correctly on different machines and broswers. This is important because it makes sure that users aren't given something that is broken.
